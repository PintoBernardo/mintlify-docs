# Extensions

All extension endpoints require `Authorization: Bearer <token>` and level 0 permission.

Writable fields:

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| type | string | yes | One of: `chan_sccp`, `chan_sip`, `chan_pjsip`, `custom` |
| extension | string | yes | Must be unique |
| name | string | yes | |
| password | string | yes | Will be bcrypt-hashed |
| description | string | no | |
| voicemail | boolean | no | Default: `false` |
| voicemail_passcode | string | no | Required if voicemail is `true` |
| is_active | boolean | no | Default: `true` |

### Request

```json

  "extension": "200",
  "name": "Sales",
  "description": "Sales team line",
  "password": "extPass123",

  "voicemail_passcode": "4321",
```
