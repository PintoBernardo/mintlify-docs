---
title: "Extensions"
description: "Extension CRUD — create, read, update, delete phone extensions"
---

All extension endpoints require level 0 permission and a Bearer token.

### Writable fields

| Field | Type | Required | Default |
|-------|------|----------|---------|
| type | string | yes | — |
| extension | string | yes | — |
| name | string | yes | — |
| password | string | yes | — |
| description | string | no | `null` |
| voicemail | boolean | no | `false` |
| voicemail_passcode | string | no | `null` |
| is_active | boolean | no | `true` |

`type` must be one of: `chan_sccp`, `chan_sip`, `chan_pjsip`, `custom`.
